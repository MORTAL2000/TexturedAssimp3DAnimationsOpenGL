to run demo you need animation, for example

1. download https://www.dropbox.com/s/mimglhfvlqdnelf/pers.zip?dl=0 
2. place it for example at c:/pers
3. replace 
AnimationController controller("C:/Dropbox/Projects/Moon/pers/walking.dae")
with 
AnimationController controller("C:/pers/walking.dae")
4. run
5. use numpad keys 1,2,3 to switch animation

there are no shaders, it just the simplest way to use assimp
